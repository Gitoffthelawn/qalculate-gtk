Lös differentialekvation

dsolve(Ekvation[, Initialt villkor: funktionsvärde (y)][, Initialt villkor: parametervärde (x)])

Löser en differentialekvation och returnerar värdet på y(x). Derivatan i ekvation måste vara i formatet diff(y; x). Enbart första ordningens differentialekvationer stöds för nuvarande.

Arguments. 

  • Ekvation: a free value

  • Initialt villkor: funktionsvärde (y): a free value (optional, default: undefined)

  • Initialt villkor: parametervärde (x): a free value (optional, default: 0)

Talbas

bas(Nummer, Bas[, Uppsättning siffror][, Omvänd omvandling])

base

Returnerar ett värde från ett uttryck med angiven talbas. Talbaser mellan -62 och 62 har fullt stöd för matematiska uttryck (inklusive operatorer och funktioner). För övriga talbaser omvandlas angivet uttryck till ett enda nummer.

Talbaser ≤ 36 används siffror 0-9 och A-Z (ej skiftlägeskänsligt).

Talbaser mellan 37 och 62 använder skiftlägeskänsliga bokstäver (0-9, A-Z, a-z) som siffror ('Z' motsvarar 61).

Talbaser över 62 använder Unicode-tecken som siffror, med teckenkoden som värde ('0' motsvarar 48). I detta kan '\' följt av ett nummer användas istället (t.ex. '\0' = 0, '\523' = 523, '\x7f' = 127).

Negativa talbaser använder samma siffror som motsvarande positiv talbas och sifforna som används för talbaser som inte är heltal avgörs genom att avrunda bort från noll. Talbaser som inte är reella tal använder som förval siffrorna 0-9 och A-Z.

Uppsättningen siffror som används kan väljas manuellt genom att använda funktionens tredje parameter (är 0 som förval, för automatiskt val). Ange som 1 för siffror 0-9 och A-Z, 2 för 0-9, A-Z och a-z, 3 för Unicode-siffror och 4 för textnummer, eller ange en textsträng med alla siffror placerade i stigande ordning (t.ex. "0123456789") och eventuellt åtskilda av semikolon (för att möjliggöra flera ekvivalenta siffror, t.ex. "0;aA1;bB2;cC3"). När uppsättningen siffror väljs manuellt, omvandlas alltid det angivna uttrycket till ett enda nummer.

Arguments.

  • Bas: a free value

  • Uppsättning siffror: a text string (optional, default: 0)

  • Omvänd omvandling: a boolean (0 or 1) (optional, default: 0)
